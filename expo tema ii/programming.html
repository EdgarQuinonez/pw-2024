Scripting Tags Overview

<script>

The script tag is used to define client-side JavaScript. It is used to embed or reference executable code.

Important Attributes:

  • src: Specifies the external file containing the script.
  • async: Loads the script asynchronously, without blocking the page.
  • defer: Delays script execution until after the document has been parsed.

<noscript>

The noscript tag defines alternate content to be displayed if JavaScript is disabled in the user's browser.

<embed>

The embed tag is used to embed external resources such as media, plugins, or content from another site. It is commonly used for multimedia like audio and video.

Important Attributes:

  • src: Specifies the source of the embedded resource.
  • type: Defines the MIME type of the embedded content.
  • width and height: Set the size of the embedded content.

<applet>

The applet tag is used to embed Java applets (small applications) in a webpage. It is now deprecated in favor of object and embed for security reasons.

Important Attributes:

  • code: Specifies the applet class file to be loaded.
  • width and height: Define the size of the applet area.
  • archive: Points to an archive (usually a JAR file) containing the applet class.

<object>

The object tag is used to embed multimedia (like images, videos, audio, or PDF files) or interactive content (like Java applets or Flash) into a webpage.

Important Attributes:

  • data: Specifies the URL of the resource to be embedded.
  • type: Defines the MIME type of the resource.
  • width and height: Set the size of the embedded content.

<param>

The param tag defines parameters for plugins or external content used within object or applet elements.

Important Attributes:

  • name: The name of the parameter.
  • value: The value of the parameter being passed.
